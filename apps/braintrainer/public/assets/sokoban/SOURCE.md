# Sokoban image assets

The player, crate, and target PNGs in this directory come from Kenney's
Sokoban asset pack:

- Official pack: https://www.kenney.nl/assets/sokoban
- Download mirror: https://opengameart.org/content/sokoban-100-tiles

The source pack is licensed under Creative Commons Zero (CC0 1.0). The
original `License.txt` is preserved as `LICENSE.txt` in this directory.
